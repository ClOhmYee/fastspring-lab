# FastAPI 기반 서버 구성

## 동기

1. 이전에 Python을 적극 활용하여 다양한 프로젝트를 진행했었으나, 프레임워크(Django, FastAPI 등)를 활용한 경험은 없음

2. 과제를 수행하며 단일 책임 원칙(SRP) 등 객체 지향 프로그래밍(OOP)을 최대한 고려한 프로젝트를 진행하는 것이 중요하다고 생각함

3. 이전에 진행하고자 계획했지만 시간 상 진행하지 못 한 개인 프로젝트의 축소판으로 계획하여 이어 도전하고자 함

## 기능

1. 기본적인 서버를 구축

2. Spring Boot 서버로부터 데이터를 받고 그에 맞는 값을 전송
